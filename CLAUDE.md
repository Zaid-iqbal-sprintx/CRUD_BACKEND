# CLAUDE.md

Project guidance for Claude Code working in this repository.

## About

Task manager REST API — a teaching project for API thinking (payloads, status
codes, validation, database structure). Node.js + Express + MongoDB Atlas.

## Rules

See @AGENTS.md for the mandatory git rules. In short: **never commit, never push,
never force-push, never change the user's branch, and never offer to do so.**
Make file changes only and leave them in the working tree for the user to review
and commit themselves.

- **NEVER change the user's branch** — no creating, switching, renaming,
  checking out, or deleting branches. Stay on the branch the user is on.
